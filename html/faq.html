How do I add a new dog to the system?

  • Click on Add New Dog Profile Tab.
  • Answer all pre selected questions.
  • Questions will cover: Dog Name, Unique ID, Breed, Birth Date, Mother's Name, Father's Name and Bloodline.
  • There are 2 sections that will require some history for completion. Medical Information Regaring Bloodline. and Behavioral Information Regarding Bloodline.

    • How do I view a specific Dog?

  • When Using Desktop
  • Hover mouse overSelect Dog Profile Tab.
  • Option1: Use search bar and start typing dog's name. Then select name from the list
  • Option2: Scroll down and select dog's name
  • When Using Mobile
  • Click on Burger menu Icon. This can be found top right hand side of screen.
  • Click on Select Dog Profile
  • Option1: Use search bar and start typing dog's name. Then select name from the list
  • Option2: Scroll down and select dog's name
